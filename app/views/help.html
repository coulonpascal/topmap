Click on the 'Add WMS/WMTS Layer' button on the add layer toolbar, or select the option from the Layer -> Add Layer menu

Click on the 'New' button to add a new WMS server to QGIS

Note: You will only have to do this step once, if you have already added the WMS server continue to step 4

  1. Add a name for this WMS connection, this can be anything
  2. Add the Get Capabilities URL of the server you wish to connect to, you can find this by clicking on the layer on the main screen or by clicking the button while viewing the layer on the map screen

Select the WMS server you wish to connect to from the drop down list and then click the connect button

Select the layer you wish to display from the list that appears and click the add button to add the layer into your current view
